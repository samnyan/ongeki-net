オプション設定
おすすめ設定
ゲーム設定
スピード設定
スピードを調整します　1～20(SONIC)から選択できます
ミラー設定
ON にすると、ノーツ表示が左右反転します
判定タイミング調整A
判定のタイミングを調整します　-2.0～+2.0から選択できます
判定タイミング調整B
ノーツの判定タイミングを調整します -2.0～+2.0から選択できます
バトル中断設定
テクニカルスコアが設定したランクに到達しない時点で、バトルを中断します
ステルスフィールド
ONにすると、フィールドが表示されなくなります
サウンド設定
TAP成功音
TAPノーツの効果音を変更できます
成功音VOL(TAP)
TAPノーツの成功音VOLを調整します
成功音VOL(CR-TAP)
CR-TAPノーツの成功音VOLを調整します
成功音VOL(HOLD)
HOLDノーツの成功音VOLを調整します
成功音VOL(SIDE)
SIDEノーツの成功音VOLを調整します
成功音VOL(FLICK)
FLICKノーツの成功音VOLを調整します
ガイド音VOL
ガイド音VOLを調整します
BELL音VOL
BELLを取った時の効果音の音量を調整します
敵弾音VOL
敵の弾をギリギリで避けた時の効果音の音量を調整します
減点カウント音VOL
「バトル中断設定」がONの時、減点発生時の効果音の音量を調整します
ダメージ音VOL
ダメージを受けた時の効果音の音量を調整します
背景表示設定
フィールドの明るさ
フィールドの明るさを調整します　-5(黒)～0(標準)の6段階から選択できます
レーンの明るさ
レーンの明るさを調整します　-3(薄い)～+3(濃い)の7段階から選択できます
SIDEレーンの明るさ
SIDEレーンの明るさを調整します　-3(薄い)～+3(濃い)の7段階から選択できます
レーンカラー
レーンの色配置を変更できます
SIDEカラー
SIDEレーンの色配置を変更できます
被弾演出
ON にすると、弾を受けた時の画面演出が発生します
キャラ座標表示
ON にすると、センターキャラクターの前方に座標線を表示します
判定表示設定
PLATINUM判定表示
ONにすると、PLATINUM BREAKの判定を表示します
※テクニカルチャレンジの対象楽曲でのみ表示されます
判定表示設定
フィールドに表示する判定/COMBO情報を設定します
判定表示位置
判定が表示される位置を設定します　天上/天/上/中/下から選択できます
判定詳細表示：CRITICAL
ONにすると、CRITICAL BREAKの判定がベストタイミングに対して「FAST(早い)/ LATE(遅い)」を表示します
※テクニカルチャレンジの対象楽曲でのみ表示されます
判定詳細表示：BREAK
ON にすると、BREAKの判定がベストタイミングに対して「FAST(早い)/LATE(遅い)」を表示します
判定詳細表示：HIT
ON にすると、HITの判定がベストタイミングに対して「FAST(早い)/LATE(遅い)」を表示します
その他の設定
マッチング通知
ON にすると、店内マッチングができます
プレイヤーレベル表示
ON にすると、ゲームプレイ中に、プレイヤーレベルを表示します
レーティング表示
ON にすると、ゲームプレイ中に、レーティングを表示します
バトルランク表示
ON にすると、ゲームプレイ中に、バトルランクとＢＰ(バトルポイント)を表示します
